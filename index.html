    Инструкция

    1. Создайте код — нажмите «Создать новый код». Скопируйте полученный 6-значный код.
    2. Загрузите файл — введите код, выберите файл (можно перетащить) и нажмите «Загрузить».
    3. Передайте код — отправьте код получателю любым способом.
    4. Получатель скачивает — вводит код в поле «Список файлов», затем вводит код и имя файла в поле «Скачать» и нажимает кнопку.
    При первом посещении может появиться страница подтверждения localtunnel. Нажмите «Перейти на страницу подтверждения», введите IP-адрес, показанный там, и нажмите «Continue». После этого сервис будет работать 7 дней без подтверждения.

    Требуется подтверждение

    Сервис localtunnel просит подтвердить ваш IP-адрес для предотвращения злоупотреблений.

    Что сделать:
    1. Нажмите «Перейти на страницу подтверждения».
    2. Введите IP-адрес, который показан на странице, и нажмите «Continue».
    3. Закройте вкладку и вернитесь сюда, затем нажмите «Повторить запрос».

    После подтверждения страница не будет показываться 7 дней.
